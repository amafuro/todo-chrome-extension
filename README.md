# 階層型TODOリスト Chrome拡張機能

親子関係を持つTODOを管理できる高機能なChrome拡張機能です。

## 主な機能

### ✅ 実装済み機能

1. **親子関係のあるTODO管理**
   - 各TODOは子タスクを持つことができます
   - 階層構造を視覚的に表示
   - 子タスクの展開/折りたたみが可能

2. **ドラッグ&ドロップによる並べ替え**
   - TODOをドラッグして位置を変更できます
   - 親TODOをドラッグすると、子TODOもまとめて移動
   - 異なるセクション間でも移動可能

3. **ハイパーリンク対応**
   - テキストに含まれるURLを自動的にリンク化
   - リンクをクリックすると新しいタブで開きます

4. **期限日管理**
   - 各TODOに期限日を設定可能
   - 期限日が近いタスクは色分けで表示
   - 期限切れタスクは警告色で表示

5. **複数セクション管理**
   - プロジェクトや用途別にセクションを分けて管理
   - セクション名は自由に変更可能
   - セクションごとに独立したTODOリスト

6. **データの永続化**
   - すべてのデータはChrome Storageに自動保存
   - ブラウザを閉じても情報は保持されます

## インストール方法

1. このフォルダ全体を任意の場所に保存します

2. Chromeブラウザを開き、以下のURLにアクセスします：
   ```
   chrome://extensions/
   ```

3. 右上の「デベロッパーモード」をONにします

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. このフォルダ（`todo-chrome-extension`）を選択します

6. ツールバーに📝アイコンが表示されたら完了です

## 使い方

### 基本操作

1. **拡張機能を開く**
   - ツールバーの📝アイコンをクリック

2. **TODOを追加**
   - 各セクションの「+ TODO追加」ボタンをクリック
   - 編集アイコン（✏️）をクリックしてテキストを入力

3. **TODOを編集**
   - ✏️アイコンをクリック
   - モーダルで内容を編集
   - 「保存」をクリック

4. **子TODOを追加**
   - 親にしたいTODOの「+」ボタンをクリック
   - 子タスクが階層表示されます

5. **TODOを完了**
   - チェックボックスをクリック
   - 完了したTODOは取り消し線で表示

6. **期限日を設定**
   - 日付アイコンをクリックして日付を選択
   - 期限日が近いと色が変わります

7. **TODOを並べ替え**
   - TODOをドラッグして好きな位置に移動
   - 親TODOを移動すると子TODOも一緒に移動

8. **セクションを追加**
   - 「+ セクション追加」ボタンをクリック
   - セクション名を入力

### 便利な使い方

- **URLの自動リンク化**
  - テキストに `https://example.com` などのURLを入力すると、自動的にクリック可能なリンクになります

- **階層構造の活用**
  - 大きなタスクを親TODOに
  - 具体的な作業を子TODOに
  - 最大で何階層でも作成可能

- **期限日の色分け**
  - 期限日なし: 通常表示
  - 期限日あり: 黄色背景
  - 期限切れ: 赤色背景（未完了の場合）

## 技術仕様

- **Manifest Version**: 3
- **Permissions**: storage（データの永続化）
- **Storage**: chrome.storage.local
- **Frameworks**: Vanilla JavaScript（フレームワーク不使用）

## トラブルシューティング

### 拡張機能が読み込めない

- manifest.jsonのパスが正しいか確認
- デベロッパーモードがONになっているか確認
- エラーメッセージを確認して対処

### データが保存されない

- Chrome Storageの権限が正しく設定されているか確認
- ブラウザのストレージ制限に達していないか確認

### ドラッグ&ドロップが動作しない

- ブラウザを再起動してみる
- 拡張機能を再読み込みしてみる
