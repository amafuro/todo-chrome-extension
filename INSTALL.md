# Chrome拡張機能インストールガイド

このガイドでは、階層型TODOリストChrome拡張機能のインストール方法を説明します。

## ステップ1: Chrome拡張機能ページを開く

1. Chromeブラウザのアドレスバーに以下を入力します：
   ```
   chrome://extensions/
   ```

2. または、Chromeメニュー（右上の3点アイコン）→「拡張機能」→「拡張機能を管理」でも開けます。

## ステップ2: デベロッパーモードを有効化

1. 拡張機能ページの右上にある「デベロッパーモード」のトグルをONにします

   ![デベロッパーモード](https://via.placeholder.com/600x100/667eea/ffffff?text=デベロッパーモード：ON)

2. トグルをONにすると、新しいボタンが表示されます

## ステップ3: 拡張機能を読み込む

1. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック

2. ファイル選択ダイアログが開きます

3. `todo-chrome-extension` フォルダを選択します
   - **重要**: フォルダ自体を選択してください（中のファイルではありません）

4. 「選択」または「開く」をクリック

## ステップ4: 拡張機能が読み込まれたことを確認

拡張機能ページに「階層型TODOリスト」が表示されていれば成功です！

表示される情報：
- 📝 アイコン
- 「階層型TODOリスト」という名前
- バージョン: 1.0.0
- 説明: 親子関係を持つTODOを管理できる拡張機能

## ステップ5: ツールバーにピン留めする（推奨）

1. Chromeツールバー右上のパズルアイコン（拡張機能アイコン）をクリック

2. 「階層型TODOリスト」の横にあるピンアイコンをクリック

3. これで、ツールバーに📝アイコンが常に表示されます

## 使い始める

1. ツールバーの📝アイコンをクリック

2. ポップアップが開きます（600x600ピクセル）

3. 「+ TODO追加」ボタンをクリックして、最初のTODOを作成しましょう！

## トラブルシューティング

### エラー: "Manifest file is missing or unreadable"

→ フォルダの選択が間違っています。`manifest.json`が含まれているフォルダを選択してください。

### エラー: "Manifest version 2 is deprecated"

→ この拡張機能はManifest V3を使用しているため、このエラーは発生しません。

### 拡張機能が表示されない

→ ページを更新（F5またはCmd+R）してみてください。

### アイコンがツールバーに表示されない

→ パズルアイコンをクリックして、ピン留めしてください。
